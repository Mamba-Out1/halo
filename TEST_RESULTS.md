### 方法 3: 添加调试日志

在测试代码中添加日志输出：

```java
@Test
void shouldDeletePostViaApi() {
    var createdPost = createTestPost("待删除文章", "delete-test");
    
    // 添加日志
    System.out.println("Created post: " + createdPost.getMetadata().getName());
    
    webTestClient
        .delete()
        .uri("/apis/content.halo.run/v1alpha1/posts/{name}",
                createdPost.getMetadata().getName())
        .exchange()
        .expectStatus().isOk();
    
    // 添加日志
    System.out.println("Delete request completed");
    
    // 验证已删除
    webTestClient
        .get()
        .uri("/apis/content.halo.run/v1alpha1/posts/{name}",
                createdPost.getMetadata().getName())
        .exchange()
        .expectStatus().isNotFound();
}
```
